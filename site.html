Agora vamos rever algumas tags e entender para o que elas servem

A tag "i" tem como função principal deixar os termos em itálico, EX: bom dia

A tag "ol" gera uma lista ordenada EX:

  1. bom dia
  2. ola
  3. tchau

A tag "ul" representa uma lista não ordenada EX:

  • arroz
  • batata
  • abobora

A tag "li" é utilizada para inclementar um item na lista

A tag "dl" é utilizada para definir uma lista de descrição

A tag "dt" identifica um termo na lista de definição EX:

A tag "dd" é utilizada para descrever um termo/nome em uma lista de descrição

exemplo dessas tags "dl","dt","dd" sendo utilizadas em conjunto :

lamborghini
é um carro super espotivo
